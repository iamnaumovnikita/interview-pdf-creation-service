**Interview Instructions: PdfCreation service**

**Project Overview:**  
Develop an application with an Angular 19 frontend and a .NET 8 backend API to enable users to download PDF maps of cities.

**Tasks:**

1. Run the existing frontend and backend applications.
2. Debug and fix the integration between the frontend and backend.
3. Implement a feature to allow users to download PDF maps for multiple cities in a single ZIP archive.
4. Suggest improvements to the current solution.

Feel free to ask questions and use any tools you prefer during the interview. There are no restrictions.
