# Interview.PdfCreation

## Project Overview
This application consists of an Angular 19 frontend and a .NET 8 backend API.
The primary purpose is to enable users to download PDF maps of cities.

## Interview Tasks
1. Run the frontend and backend application
2. Debug and fix the integration between the frontend and backend
3. Implement a new feature: Allow users to download PDF maps for multiple cities packaged in a single ZIP archive
---
4. Suggest any improvements you would make to the current solution

Feel free to ask questions and use any tools you prefer during the interview. There are no restrictions.
