# Interview Frontend (React)

This is the React version of the PDF creation service frontend.

## Development server

Run `npm start` to start the development server. Navigate to `http://localhost:4201/`. The application will automatically reload if you change any of the source files.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `build/` directory.

## Running tests

Run `npm test` to execute the unit tests via Jest and React Testing Library.
